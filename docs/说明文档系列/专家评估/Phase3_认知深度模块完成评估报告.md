## 变更摘要

所有改动集中在 `api/cpi/narrative_engine.py`（CPI 解耦的价值 — `airi_bridge.py` 零修改）

| #   | 模块        | 改动                                                                                                    |
| --- | ----------- | ------------------------------------------------------------------------------------------------------- |
| 1   | 感知聚合器  | 新增 `get_spatial_manager()` 空间位置感知，prompt 展示位置                                              |
| 2   | 语义记忆    | `recall_memories` 支持关键词相关性排序（获取 3x 记忆 → 评分 → 返回 top-k）                              |
| 3   | 意识流触发  | `process_stimulus` 完整实现：构建角色档案 → 调用 `ucs.process_consciousness_stream()` → 解析 4 阶段输出 |
| 4   | Prompt 重构 | 新增 RGES 反思洞察区域（`# 近期反思`），位置信息展示                                                    |
| 5   | RGES 闭环   | `submit_experience` 返回即时反思结果 `insights []`                                                      |
| 6   | L1-L3 升级  | `_extract_key_facts()` 规则引擎（名字/偏好/事件识别）→ 关键信息存入 L2                                  |
