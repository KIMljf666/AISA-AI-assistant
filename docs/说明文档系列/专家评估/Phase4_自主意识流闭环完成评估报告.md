## 变更摘要

### `airi_bridge.py` — consciousness_loop 重写

| 步骤            | Before                                     | After                                                             |
| --------------- | ------------------------------------------ | ----------------------------------------------------------------- |
| Step 1 环境感知 | ❌ 无                                       | ✅ CPI `get_perception()` → 广播 Layer 0 事件                      |
| Step 2 意识流   | `_try_get_consciousness_stream()` 静态读取 | ✅ CPI `process_stimulus()` → UCS 4 阶段处理                       |
| Step 3 主动开口 | ❌ 无                                       | ✅ `_estimate_importance()` > 0.7 时广播 `proactive_dialogue` 事件 |
| Step 4 情绪追踪 | ❌ 无                                       | ✅ 跨周期情绪对比 → 广播 `emotion` 变化事件                        |
| 需求广播        | `json.dumps(needs)[:100]` 原始数据         | ✅ `needs_to_natural_language()` 自然语言                          |
| 数据来源        | `_try_get_*` 直接 core/ 调用               | ✅ CPI provider 统一调用                                           |

### 新增函数

| 函数                            | 说明                                          |
| ------------------------------- | --------------------------------------------- |
| `_estimate_importance(content)` | 估算意识流内容重要度 (0-1)，关键词 + 长度加权 |

## 意识流闭环工作流

```
┌─────────────────────────────────────┐
│  consciousness_loop (每 30s)        │
│                                     │
│  1. get_perception() → Layer 0 广播  │
│  2. get_psychological_state()       │
│  3. process_stimulus() → 4阶段广播   │
│     ├─ perception  → Layer 0        │
│     ├─ processing  → Layer 2        │
│     ├─ decision    → Layer 6        │
│     └─ execution   → Layer 8        │
│  4. _estimate_importance()          │
│     └─ > 0.7 → proactive_dialogue   │
│  5. 情绪对比 → emotion 变化广播      │
│  6. 需求自然语言广播                 │
└─────────────────────────────────────┘
       ↓ WebSocket
┌─────────────────────────────────────┐
│  CognitiveBubble.vue                │
│  实时展示所有认知事件                │
│  proactive_dialogue 可触发角色主动说 │
└─────────────────────────────────────┘
```
