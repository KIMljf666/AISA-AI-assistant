## 二、Phase A: 定量基线测试 (A/B 对照)

### 设计

```
┌───────────────┬──────────────────────┬──────────────────────┐
│   场景 (20轮) │  Baseline (纯LLM)    │  Full Pipeline       │
│               │  只有角色人设prompt   │  感知+记忆+UCS+RGES  │
├───────────────┼──────────────────────┼──────────────────────┤
│  日常闲聊      │        A1            │        B1            │
│  情感支持      │        A2            │        B2            │
│  知识探索      │        A3            │        B3            │
│  角色世界观    │        A4            │        B4            │
└───────────────┴──────────────────────┴──────────────────────┘
```

### Baseline 控制

```python
# Baseline: 关闭所有增强模块
BASELINE_CONFIG = {
    "enable_perception": False,    # 不注入时间/天气/节日
    "enable_memory": False,        # 不检索记忆
    "enable_consciousness": False, # 不调用 UCS
    "enable_rges": False,          # 不触发反思
    "enable_four_layers": False,   # 不注入四层心理
    "enable_key_facts": False,     # 不提取关键信息
}

# Full Pipeline: 全部开启 (当前默认)
FULL_CONFIG = {
    "enable_perception": True,
    "enable_memory": True,
    "enable_consciousness": True,
    "enable_rges": True,
    "enable_four_layers": True,
    "enable_key_facts": True,
}
```

### 预设对话脚本

```yaml
# 场景 1: 日常闲聊 (20轮)
turns:
  - "嗨，你好呀"
  - "最近有什么有趣的事情吗？"
  - "哦对了，我叫小明"               # ← 测试记忆: 后面看是否记得
  - "你平时都做些什么？"
  - "我特别喜欢看科幻电影"            # ← 测试提取: 偏好
  - "你有看过《星际穿越》吗？"
  - "今天天气真好"
  - "对了你还记得我的名字吗？"          # ← 测试回忆: 第3轮说的名字
  - "我觉得你特别有想法"
  - "最近工作压力好大"                 # ← 测试情感: 负面情绪触发
  # ... 20轮

# 场景 2: 情感支持 (20轮)
turns:
  - "我最近心情特别不好"
  - "工作上被老板批评了"
  - "感觉自己什么都做不好"
  - "你觉得我是不是很没用？"             # ← 测试: 角色是否给出共情
  - "谢谢你听我说这些"
  # ...
```

### 自动评分指标

| 指标 | 计算方法 | 目标 | 权重 |
|------|---------|------|------|
| 😊 **角色一致性** | Judge LLM 评估回复是否符合角色 MBTI/人设 | ≥ 0.8 | 25% |
| 🧠 **记忆引用率** | 后10轮中引用前10轮信息的次数 / 总轮数 | ≥ 30% | 25% |
| 💬 **回复质量** | Judge LLM 评估回复深度/相关性/情感匹配 (1-5) | ≥ 3.5 | 25% |
| 🔄 **情绪连续性** | 相邻轮次情绪表达的余弦相似度 | ≥ 0.6 | 15% |
| 📝 **自主表达** | 回复中包含非用户提及内容的比例 (角色主动分享) | ≥ 20% | 10% |

### 实现方式

```bash
# 运行 A/B 测试
python tests/evaluation/run_ab_test.py \
  --character saihisis \
  --scenario casual_chat \
  --turns 20 \
  --mode baseline  # 或 full_pipeline

# 输出: data/evaluation/{timestamp}_ab_results.json
```

## 三、Phase B: 长会话深度测试 (50+ 轮)

### B1: 记忆累积与检索

| 轮次 | 动作 | 验证点 |
|------|------|--------|
| 第5轮 | 告知角色 "我叫小明" | key_facts 是否提取到 name |
| 第10轮 | 告知 "我喜欢猫，养了一只叫橘子" | 偏好是否存储 |
| 第15轮 | 聊别的话题 | 记忆不应干扰 |
| 第25轮 | 问 "你还记得我养的宠物吗？" | **向量检索是否找到第10轮** |
| 第35轮 | 问 "我叫什么？" | **L2记忆是否正确回忆** |
| 第50轮 | 问 "我们最早聊了什么？" | **长距离记忆检索** |

### B2: 反思与四层心理演化

| 轮次 | 操作 | 验证点 |
|------|------|--------|
| 第1-10轮 | 正常聊天 | quality ≈ 0.7, 无反思 |
| 第11轮 | 降低 LLM temperature = 0.1 | 回复变刻板 |
| 第15轮 | 检查 LifeMoment | quality 应下降 |
| 第20轮 | 检查反思记录 | 应有 1+ 反思 |
| 第25轮 | 检查 four_layers.state | **应有变化** |
| 第30-50轮 | 恢复正常 | state 是否恢复？ |

### B3: RGES 目标系统

| 操作 | 验证 |
|------|------|
| 初始设定 3 个目标 | GoalAwareness 列表 |
| 连续对话 30 轮 | LifeMoment 中 related_goal_ids 是否被填充 |
| 检查 GoalEvolution | 目标是否有 strengthen/weaken/reframe |

### B4: 恢复测试

| 步骤 | 说明 |
|------|------|
| 1. 对话 20 轮 | 积累数据 |
| 2. 停止服务 | 模拟重启 |
| 3. 重启并继续 | 验证记忆/LifeMoment 持久化 |
| 4. 问 "你还记得我们之前聊的吗？" | **持久化验证** |

## 四、Phase C: 人类主观评估

### 评估者要求

- 3-5 名内部测试者
- 每人进行 2 次对话: 一次 Baseline, 一次 Full Pipeline (盲测, 随机顺序)
- 每次 15-20 轮

### 评分表

| 维度 | 说明 | 1 | 2 | 3 | 4 | 5 |
|------|------|---|---|---|---|---|
| 🧠 自主性 | 角色是否有"自己的想法"？ | 完全机械 | 偶尔有 | 一般 | 较明显 | 非常有灵魂 |
| 📝 记忆力 | 角色记得之前的对话吗？ | 完全不记得 | 偶尔提及 | 一般 | 大部分记得 | 完美 |
| 🎭 一致性 | 角色表现是否前后一致？ | 人格分裂 | 偶尔跑偏 | 基本一致 | 高度一致 | 完美 |
| 💗 情感 | 角色的情感表达自然吗？ | 冷冰冰 | 生硬 | 一般 | 自然 | 令人动容 |
| ⭐ 沉浸感 | 你觉得在和真人对话吗？ | 完全像AI | 大部分像AI | 一半一半 | 大部分像真人 | 分不清 |

## 五、Phase D: OpenClaw 自动化评测监控

> 详见系统概览文档中的 OpenClaw 评测 Agent 方案

### 核心理念

```
OpenClaw Mission Control
     ↓ (cron: 每小时)
读取 LifeMoment JSONL + Memory JSONL
     ↓
Judge LLM 评分 (角色一致性/记忆/质量)
     ↓
写入 SQLite (evaluation_scores)
     ↓
Mission Control Dashboard 可视化
     ↓
异常告警: quality 移动平均 < 0.6
```

### Cron Job 配置

```json
{
  "name": "consciousness-eval-hourly",
  "schedule": "0 * * * *",
  "model": "bailian/qwen3.5-plus",
  "prompt": "读取最近10条LifeMoment, 评估角色一致性/记忆引用/回复质量, 输出JSON评分",
  "output": "data/evaluation/hourly_scores.jsonl",
  "enabled": true
}
```
